# Text Behind: A Simple Web App for Overlaying Text Behind Images

Text Behind is a web application that enables users to add text behind images or objects in their photos. It is completely free and available for unlimited use.

## About the Project

Designed as a creative tool, Text Behind allows users to seamlessly overlay text on images. Whether for memes, captions, or artistic projects, the app simplifies the process with an intuitive interface.

## Features

- **Unlimited Use:** No restrictions – users can generate as many text overlays as they like.
- **User-Friendly Interface:** Easily upload images and customize text.
- **Customization Options:** Adjust text size, style, and positioning to achieve the perfect visual effect.

## Live Demo

Explore the live version of Text Behind here:  
[Live Demo](https://textbehind.robinweb.site/)

## How It Works

1. **Upload an Image:** Users begin by selecting an image for customization.
2. **Add Text:** They can input text that will be placed behind the image’s objects.
3. **Adjust & Customize:** Text size, style, and placement can be fine-tuned.
4. **Save & Share:** Once satisfied with the result, users can save or share their creation.

## Technologies Used

- **Frontend:** Next.js, Tailwind CSS, Shadcn-UI
- **Backend:** Next.js Server Actions
- **Additional Tools:** "html2canvas" npm package
